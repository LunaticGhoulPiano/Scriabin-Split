# Scriabin-Split
My first ergo split keyboard.
Alexander Nikolayevich Scriabin (Александр Николаевич Скрябин) is a pianist famous for his small hands. This project use his name as a reference of my basic requirement of my own keyboard.

## Notice
Current works are all working on [ceoloide's website](https://ergogen.ceoloide.com/). If you want to run locally, make sure you know how to configure footprints and the pcb block.

## Tools
- [Ergogen](https://github.com/ergogen/ergogen) - [Unofficial version](https://ergogen.ceoloide.com/)
- [OpenSCAD](https://openscad.org/)
- [Kicad](https://www.kicad.org/)
